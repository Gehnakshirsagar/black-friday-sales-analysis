# Black Friday Sales Analysis

Exploring Black Friday shopping patterns across age groups and gender using Tableau.

## What I Did
I analyzed a dataset of 550,000+ Black Friday transactions to understand customer spending behavior.

This was my first time using Tableau so I had to do some research along the way to figure out how to build dashboards. I kept it simple with 2 clear charts that tell the most important story in the data.

## Dashboard Preview
![Dashboard](dashboard.png)

## What the Dashboard Shows
- Male customers spend around 3x more than female customers on Black Friday
- Customers aged 26-35 are by far the biggest spenders, followed by the 36-45 age group

## Dataset
Black Friday Sales dataset from Kaggle.
Download train.csv and place it in the data folder to explore the data yourself.

## Tools Used
- Tableau for the dashboard
