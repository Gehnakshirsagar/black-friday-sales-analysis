# black-friday-sales-analysis
Exploring Black Friday shopping patterns using Tableau and sales data

Exploring Black Friday shopping patterns across age groups, gender, and product categories using Tableau.

## What I Did
I analyzed a dataset of 550,000+ Black Friday transactions to understand customer spending behavior.

This was my first time using Tableau so I had to do some research along the way to figure out how to build dashboards. I built 4 charts to visualize the findings.

## What the Dashboard Shows
- Male customers spend significantly more than female customers
- Customers aged 26-35 are the biggest spenders on Black Friday
- City category B has the highest total purchases
- Product category 1 is the most purchased category

## Dataset
Black Friday Sales dataset from Kaggle.
Download train.csv and place it in the data folder to explore the data yourself.

## Tools Used
- Tableau for the dashboard
- Python and SQL for initial data exploration
